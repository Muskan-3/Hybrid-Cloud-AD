Hybrid Cloud - Azure Lab Experiments

#Overview

This repository contains a collection of Microsoft Azure cloud computing experiments performed as part of a hybrid cloud lab. It demonstrates practical implementation of Azure services including Virtual Machines, Networking, IAM, Web Apps, and Security configurations.

#Experiments Included

1. Azure Dashboard Access

* Accessed Azure Portal and explored dashboard features
* Screenshot: `Azure_Dashboard_Access.png`

2. Create Virtual Machine (Portal, CLI & PowerShell)

* Created VM using Azure Portal
* Created VM using Azure CLI
* Created VM using PowerShell
* Screenshots:

  * `Created Virtual Machine in Azure.png`
  * `VM_Creation_CLI.png`
  * `VM_Creation_PowerShell.png`
  
3. Create Web App

* Deployed a web application using Azure App Service
* Verified deployment via default URL
* Screenshot: `my_Web_APP_Azure.png`

4. Resource Providers

* Viewed and managed Azure resource providers
* Screenshot: `Resource Providers.png`

5. Resource Lock

* Applied and removed lock on Resource Group
* Screenshots:

  * `Lock_Resource_Group.png`
  * `Released Lock.png`

6. Virtual Network Configuration

* Created Virtual Network and subnet
* Connected VM to VNet
* Screenshot: `Virtual Network and Virtual Machine Configure.png`

7.1 Network Security Group (NSG)

* Created NSG and associated with VM
* Added inbound rule (SSH - Port 22)
* Screenshot: `NSG Associated with VM.png`

 7.2 SSH Access to VM

* Connected to VM using SSH
* Executed basic Linux commands
* Screenshot: `VM is successfully accessed using SSH.png`
  
8.IAM (Identity and Access Management)

* Assigned roles using RBAC
* Added **Reader role** to user
* Screenshot: `IAM Implemented.png`
